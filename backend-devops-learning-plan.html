01
Foundations of the Web & Programming
HTTP · Networking · Python or Node.js · Linux basics
4–5 weeks
▾
How the Web Works
  • HTTP/HTTPS: request/response cycle
  • DNS, IP addresses, ports
  • TCP/IP model basics
  • Status codes: 2xx, 3xx, 4xx, 5xx
  • Headers, cookies, sessions
  • REST vs RPC vs GraphQL (overview)
Linux Command Line
  • File system navigation (ls, cd, pwd)
  • File ops: cp, mv, rm, chmod, chown
  • Processes: ps, kill, top, htop
  • Text tools: grep, sed, awk, cut
  • Bash scripting fundamentals
  • SSH, scp, rsync
Programming Fundamentals
  • Pick Python or Node.js as your language
  • Variables, types, control flow
  • Functions, modules, imports
  • Data structures: lists, dicts, sets
  • Error handling & exceptions
  • Reading/writing files, JSON, env vars
Resources The Odin Project MDN — How the Web Works Linux Command (book) CS50x — Harvard roadmap.sh/backend
02
Backend Development & APIs
REST APIs · Auth · Databases · Frameworks
5–6 weeks
▾
Building REST APIs
  • Express.js or FastAPI setup
  • Routes, controllers, middleware
  • Request validation & error handling
  • CRUD operations
  • Pagination, filtering, sorting
  • API versioning best practices
Databases
  • SQL fundamentals: SELECT, JOIN, GROUP BY
  • PostgreSQL setup & usage
  • Indexes, constraints, transactions
  • ORM: Prisma, SQLAlchemy, or TypeORM
  • Database migrations
  • NoSQL basics: MongoDB or Redis
Auth & Security
  • Hashing passwords (bcrypt, argon2)
  • JWT: signing, verifying, refresh tokens
  • OAuth2 / social login basics
  • Rate limiting & brute-force protection
  • OWASP Top 10 vulnerabilities
  • HTTPS & TLS in practice
Resources FastAPI Docs Express.js Docs PostgreSQL Tutorial JWT.io Fireship — Backend videos
03
Containers, Git & DevOps Fundamentals
Docker · Git workflows · Networking · Testing
4–5 weeks
▾
Docker
  • Images, containers & layers
  • Writing Dockerfiles
  • Multi-stage builds
  • Docker Compose for local dev
  • Volumes & bind mounts
  • Networking between containers
  • Publishing to Docker Hub / GHCR
Git & Collaboration
  • Branching strategies (Git Flow, trunk)
  • Rebase, cherry-pick, squash
  • Pull requests & code review
  • Semantic versioning & changelogs
  • Monorepos vs polyrepos
  • Git hooks with Husky
Testing & Quality
  • Unit, integration & end-to-end tests
  • Test-driven development (TDD)
  • Mocking & test doubles
  • API testing with Postman / Bruno
  • Code coverage & linting
  • Load testing with k6 or Locust
Resources Docker Docs Pro Git Book (free) TechWorld with Nana k6 Docs
04
CI/CD & Cloud Fundamentals
GitHub Actions · AWS/GCP · Terraform · Serverless
5–6 weeks
▾
CI/CD Pipelines
  • GitHub Actions: workflows & jobs
  • Build, test & lint in CI
  • Docker image build & push in CI
  • Deployment strategies: blue/green, canary
  • Secrets & environment management
  • GitLab CI or CircleCI as alternatives
Cloud Platforms
  • AWS core: EC2, S3, RDS, VPC, IAM
  • Load balancers & Auto Scaling Groups
  • Managed databases: RDS, ElastiCache
  • Object storage: S3, CloudFront CDN
  • Serverless: Lambda & API Gateway
  • Cloud cost management basics
Infrastructure as Code
  • Terraform: providers, resources, state
  • Modules & remote state (S3 backend)
  • terraform plan, apply, destroy
  • Variables, outputs & workspaces
  • AWS CDK or Pulumi as alternatives
  • Ansible for config management
Resources GitHub Actions Docs AWS Free Tier + Skill Builder Terraform Docs FreeCodeCamp DevOps course Cloud Resume Challenge
05
Kubernetes & Observability
K8s · Helm · Prometheus · Grafana · GitOps
5–6 weeks
▾
Kubernetes Core
  • Pods, Deployments, ReplicaSets
  • Services: ClusterIP, NodePort, LoadBalancer
  • ConfigMaps & Secrets
  • Persistent Volumes & PVCs
  • Namespaces, RBAC, service accounts
  • Ingress controllers & TLS
  • Horizontal Pod Autoscaler
Helm & GitOps
  • Helm charts: templates, values
  • Installing & upgrading releases
  • Helm repositories
  • GitOps with ArgoCD or Flux
  • Kustomize for environment overlays
  • EKS, GKE, or AKS managed clusters
Observability Stack
  • Prometheus: scraping, metrics, rules
  • Grafana: dashboards & alerts
  • Loki + Promtail for log aggregation
  • Distributed tracing: OpenTelemetry
  • Alertmanager & PagerDuty integration
  • SLIs, SLOs, and error budgets
Resources Kubernetes Docs TechWorld with Nana — K8s KodeKloud Helm Docs CKA Exam Prep
06
Advanced Architecture & SRE Practices
Microservices · Security · Performance · System Design
5–6 weeks
▾
Microservices & Messaging
  • Service decomposition strategies
  • Apache Kafka: topics, consumers, producers
  • Event-driven architecture patterns
  • API Gateway patterns
  • Service mesh: Istio or Linkerd
  • gRPC for internal service comms
Security & Compliance
  • Container & image scanning (Trivy)
  • Secrets management: Vault, AWS Secrets Manager
  • Network policies in Kubernetes
  • Static analysis & SAST/DAST in CI
  • Least privilege & zero-trust principles
  • Compliance: SOC 2, ISO 27001 basics
Performance & Reliability
  • Caching strategies: write-through, cache-aside
  • Database sharding & read replicas
  • CDN design & edge caching
  • Chaos engineering with Chaos Monkey
  • Runbooks & incident management
  • Capstone: full cloud-native system design
Resources Designing Data-Intensive Apps — Kleppmann System Design Primer (GitHub) Google SRE Book (free) HashiCorp Vault Docs
