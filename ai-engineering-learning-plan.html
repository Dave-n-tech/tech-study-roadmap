Python & Math Foundations
Python · NumPy · Pandas · Linear Algebra · Calculus · Statistics
5–6 weeks
▾
Python for Data & AI
  • Python data types, comprehensions, generators
  • NumPy: arrays, broadcasting, vectorization
  • Pandas: DataFrames, groupby, merging
  • Matplotlib & Seaborn for visualization
  • Jupyter notebooks & Google Colab
  • Virtual environments & dependency management
Linear Algebra
  • Vectors, matrices & tensors
  • Matrix multiplication & dot products
  • Eigenvalues & eigenvectors
  • SVD & matrix decompositions
  • Norms and distance metrics
  • Dimensionality & linear transformations
Calculus & Statistics
  • Derivatives, gradients & chain rule
  • Partial derivatives & Jacobians
  • Probability distributions: Gaussian, Bernoulli
  • Bayes' theorem & conditional probability
  • Expectation, variance, covariance
  • Maximum likelihood estimation (MLE)

Machine Learning Fundamentals
Scikit-learn · Supervised Learning · Model Evaluation · Feature Engineering
5–6 weeks
▾
Supervised Learning
  • Linear & logistic regression
  • Decision trees & random forests
  • Gradient boosting: XGBoost, LightGBM
  • Support vector machines (SVM)
  • k-Nearest Neighbors (k-NN)
  • Naive Bayes classifiers
Unsupervised & Evaluation
  • K-means & hierarchical clustering
  • PCA for dimensionality reduction
  • Anomaly detection basics
  • Train/val/test splits & cross-validation
  • Metrics: accuracy, F1, AUC-ROC, RMSE
  • Bias-variance tradeoff & overfitting
Feature Engineering & Pipelines
  • Feature scaling & normalization
  • Handling missing values & outliers
  • Encoding categorical variables
  • Feature selection & importance
  • Scikit-learn Pipeline API
  • Hyperparameter tuning: GridSearch, Optuna

03
Deep Learning
PyTorch · Neural Networks · CNNs · RNNs · Transformers
6–7 weeks
▾
Neural Network Fundamentals
  • Perceptrons & activation functions
  • Forward pass & backpropagation
  • Loss functions: cross-entropy, MSE
  • Optimizers: SGD, Adam, AdamW
  • Regularization: dropout, batch norm, weight decay
  • Learning rate scheduling
PyTorch & Architectures
  • Tensors, autograd & computation graphs
  • nn.Module, DataLoader, training loops
  • Convolutional networks (CNNs) for vision
  • Recurrent networks: LSTM, GRU
  • Transfer learning & fine-tuning
  • GPU training & mixed precision
Transformers & Attention
  • Attention mechanism & self-attention
  • Transformer architecture (encoder/decoder)
  • Positional encoding
  • BERT, GPT & ViT families
  • Hugging Face Transformers library
  • Tokenization strategies (BPE, WordPiece)

04
LLMs, RAG & Generative AI
Prompt Engineering · Embeddings · Vector DBs · Fine-tuning · Agents
6–7 weeks
▾
LLMs & Prompt Engineering
  • LLM internals: pretraining, RLHF, instruction tuning
  • Prompt engineering: zero/few-shot, CoT
  • OpenAI, Anthropic & open-source APIs
  • Structured outputs & function calling
  • Context windows & token management
  • Evaluation: BLEU, ROUGE, LLM-as-judge
Embeddings & RAG
  • Text embeddings & semantic search
  • Vector databases: Pinecone, Qdrant, pgvector
  • RAG pipeline: chunking, indexing, retrieval
  • Hybrid search: keyword + semantic
  • Reranking & context compression
  • Advanced RAG: HyDE, multi-query, RAPTOR
Fine-tuning & Agents
  • LoRA & QLoRA parameter-efficient fine-tuning
  • Dataset preparation & instruction tuning
  • LangChain & LlamaIndex frameworks
  • AI agents: ReAct, tool use, memory
  • Multi-agent orchestration
  • Guardrails, safety & output validation
Resources DeepLearning.AI Short Courses LangChain Docs LlamaIndex Docs OpenAI API Docs Hugging Face PEFT (LoRA)
05
MLOps & Production Deployment
Experiment Tracking · Model Serving · Monitoring · CI/CD for ML
5–6 weeks
▾
Experiment Tracking & Registry
  • MLflow: runs, experiments, artifact tracking
  • Weights & Biases (W&B) dashboards
  • Model versioning & registry
  • Reproducibility: seeds, configs, environments
  • DVC for data & model versioning
  • Feature stores: Feast, Tecton basics
Model Serving
  • REST APIs for model inference (FastAPI)
  • Model formats: ONNX, TorchScript, SavedModel
  • Triton Inference Server
  • vLLM & TGI for LLM serving
  • Batching, caching & latency optimization
  • AWS SageMaker / Vertex AI deployment
CI/CD for ML & Monitoring
  • ML pipelines: Kubeflow, ZenML, Prefect
  • Automated retraining triggers
  • Data drift & concept drift detection
  • Model performance monitoring
  • LLM observability: LangSmith, Helicone
  • A/B testing & shadow deployments
Resources MLflow Docs Weights & Biases Docs DVC Docs vLLM Docs Made With ML (MLOps course)

Advanced AI Systems & Architecture
Multimodal · Evals · AI Safety · Distributed Training · System Design
5–6 weeks
▾
Multimodal & Frontier Models
  • Vision-language models (CLIP, LLaVA, GPT-4V)
  • Image generation: diffusion models, SDXL
  • Audio models: Whisper, speech synthesis
  • Video understanding & generation
  • Multimodal RAG pipelines
  • Model context protocol (MCP)

  • Red-teaming & adversarial testing

  • AI regulation & compliance basics
Distributed Training & Scale
  • Data parallelism & model parallelism
  • DeepSpeed & FSDP for large model training
  • Quantization: INT8, GPTQ, AWQ
  • Flash Attention & memory optimization
  • Capstone: end-to-end AI product design
  • Cost optimization for inference at scale
